# Real-time Multi-player Terminal Game

A tic-tac-toe game but in terminal.

### Dependencies
 - Socket.io Server  
 - Socket.io Client  
 - Blessed  
 - Chalk  
 - Figlet  
 - Clear

 # Run the Game Server

 ```
 npm run server
 ```

 # Run the Game Client

 You will need two players to start a game. So open up two terminal windows and run

 ```
npm run client
 ```
